Cette liste déroulante permet d'opérer une rotation des pages, dans le sens horaire.

La valeur est exprimée en degrés. La valeur 90 tourne la page d'1/4 de tour dans le sens des aiguilles d'une montre, la valeur 270 tourne la page d'1/4 de tour dans le sens inverse des aiguilles d'une montre

La valeur par défaut est 0.

Si un fichier de test est fourni, le bouton Test permet de contrôler l'effet sur les pages du fichier.
