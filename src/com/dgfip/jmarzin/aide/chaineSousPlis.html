Cette donnée permet de préciser la chaîne de caractères qu'on doit trouver dans un document pour qu'il soit placé dans un fichier dont le nom contiendra les mots SousPlis.

Ce fichier, destiné à être envoyé à l'éditique, contiendra les versos des documents.

L'utilisation de cette donnée n'a de sens que si la donnée chaineService: est servie.

Par défaut, elle est vide, ce qui conduit à ce que le document soit toujours mis dans le fichier SousPlis.

Si un fichier de test est fourni, le bouton Test permet de vérifier les pages qui iront dans le fichier SousPlis et les pages qui iront dans le fichier Service.
