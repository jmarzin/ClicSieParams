Chaque type de document doit avoir un nom unique. Il est recommandé que le nom soit composé :
  • du nom du type d'acte auquel il correspond,
  • et éventuellement, si le type d'acte comprend plusieurs types de documents,
    • du séparateur _
    • d'un nom qui permet d'identifier chaque document pour le type d'acte.

Il est préférable qu'il soit en majuscules.
