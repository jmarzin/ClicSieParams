Ce paramètre n'a de sens que si l'adresse de destination doit être mise aux normes, c'est-à-dire si le paramètre adresseDest: est servi.

Si la case est cochée, l'adresse d'origine sera supprimée, pour ne laisser subsister que l'adresse aux normes. C'est généralement le cas quand il s'agit de replacer l'adresse du destinataire.

Par défaut, la case n'est pas cochée.

Si un fichier de test est fourni, le bouton test permet de contrôler la première page du fichier transformée conformément à ce paramètre et à tous ceux qui le précèdent.
