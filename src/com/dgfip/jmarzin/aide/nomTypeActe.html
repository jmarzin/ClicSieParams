Ce paramètre indique le type d'acte auquel le type de document appartient.

Si un nom différent de celui du type d'acte affiché est saisi, le nom doit correspondre à un type d'acte qui existe déjà. Le type de document y sera rattaché dès qu'on aura changé de type d'acte affiché (touche "suivant" ou "précédent" par exemple).
