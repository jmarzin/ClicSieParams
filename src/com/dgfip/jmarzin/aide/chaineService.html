Cette donnée permet de préciser la chaîne de caractères qu'on doit trouver dans un document pour qu'il soit placé dans un fichier dont le nom contiendra le mot Service.

Ce fichier, destiné à être conservé par le service, ne contiendra pas les versos des documents.

L'utilisation de cette donnée n'a de sens que si la donnée chaineSousPlis: est servie.

Sa valeur par défaut est null, ce qui conduit à un fichier Service vide.

Si un fichier de test est fourni, le bouton Test au regard du paramètre chaineSousPlis: permet de vérifier les pages qui iront dans le fichier SousPlis et les pages qui iront dans le fichier Service.
